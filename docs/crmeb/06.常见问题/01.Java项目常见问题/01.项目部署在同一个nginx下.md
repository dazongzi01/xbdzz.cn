### 个别开发需求需要将前端项目部署在一个nginx下使用目录区分

也就是下面这样的访问方式
WEBPC管理端         http://domain:port/admin
H5商城                     http://domain:port/h5
甚至更多。

那么基于Crmeb java版本的项目该如何实现上面的需求呢？

### 需求

在同一个nginx目录下存放admin 和 h5
admin path = admin
h5 path = h5
目录仅为实例 如果有需求请已自己的目录名称为准，这里仅做示例
