# 视频号草稿操作 **粗体请仔细阅读**

#### WEB PC端具体操作

1. > 1. 此小程序必须在WEB PC 管理端正确配置
   > 2. 营销-->视频号管理-->草稿列表-->添加基础商品 提交后微信会自动审核一般10分钟到1小时不等，审核后商品会自动上架，这里的上架并非直接视频号已经开卖，而是直播时可选的带货商品而已。
   > 3. 营销-->视频号管理-->商品列表    此列表为微信审核通过的商品，也就是可售卖商品，此时可操作上下架及删除。

#### **也就是说视频号添加商品通过草稿列表添加，上下架删除商品通过商品列表操作。**
